Rust-powered SPICE automation for Python users and systems developers.

DeSpice, packaged as pyspice-rs, builds circuits in Python, emits backend-specific netlists through a Rust intermediate representation, runs SPICE simulators, parses results, and provides reusable analog and mixed-signal design testbenches.
